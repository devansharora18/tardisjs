# Counter Demo (tardisjs)

A simple demo website for the counter use case.

## Structure

- `pages/index.tardis` — main page (`/`)
- `components/Counter.tardis` — reusable counter component example
- `tardis.config.js` — demo config

## Run

From this folder:

```bash
npx tsx ../../bin/tardis.ts dev
```

Then open `http://localhost:3001`.

## Build + Preview

```bash
npx tsx ../../bin/tardis.ts build
npx tsx ../../bin/tardis.ts preview
```
